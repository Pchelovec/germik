Программа для озвучки текста в формат mp4.
Программа позволяет создать из исходников по изображению и тексту что-то на подобии композиции.
Изображение назвать "image" и положить в папку.
Текст вписать в исходный код программы, написаной на языке python
Для запуска скачать зависимости при помощи pip.


Program for voice acting text to mp4 format.
The program allows you to create something like a composition from sources using an image and text.
The image is named "image" and placed in a folder.
Text written in program source code written in python language
To run, download the dependency using pip.
